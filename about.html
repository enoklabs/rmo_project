About This Project

Rich Media Optimization Final Project

Rich Media Optimization is an online course from Full Sail University and the goal of this final project for this course was to apply our understanding of media optimization to a real world entity.

This was accomplished through the creation of this multimedia gallery in the main page. This gallery showcase a set of various video and audio clips, provided in numerous file formats and settings. This is necessary for media elements because of the many of browsers and devices that are used nowadays to surf the internet and interact with these rich media documents.

There can be many cross-browser compatibility issues between browsers and devices because not all of the read the same file types. Our job as web developers is to provide an abundance of options in file types for the browsers to accept and understand.

The tools learned in this course were the following:

  • ➜ iMovie
  • ➜ Adobe Audition
  • ➜ Adobe Premiere Pro
  • ➜ Adobe Media Encoder
